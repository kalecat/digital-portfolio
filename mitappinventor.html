MIT App Inventor allow users to create touch-screen applications on Android operating systems through computer programming. Coding is executed thorugh a graphical interface, with simple blocks similar to Scratch. Application design is carried out by dragging components onto a virtual tablet screen.

MIT App Inventor Projects

Space Invaders



I created this app using a tutorial on the MIT App Inventor website. The platform's setup is very similar to Scratch, so it wasn't too difficult to figure out. Creating this app was fairly simple and was basically an introduction to the different components in the program, their purpose, and how to use the code. I would advise a first-time programmer to use the tutorials on the MIT App Inventor website and ensure that they understand the different components and how to use the blocks.

Jumping Girl









I created this game on my own with the help of some Youtube tutorials. This project was very frustrating, but I am proud of the final product. I think the most difficult part of creating this game was getting the girl to jump the way I wanted her to. Creating this part of the code taught me just how important every part of the code is. Another difficult part of this project was creating a scrolling background. Trying to get the code to work initially caused some glitching, but once I figured it out, getting everything else to move across the screen was fairly easy. Most of the other components of this game were easy to create because I had used them before. I would advise a first time programmer to use all of their available resources, such as youtube and other tutorials. Additionally, if something does not work, I would advise putting code in the backpack until the rest of the code works properly in order to find the dysfunctional piece of code.
