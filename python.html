Python is an interpreted, object-oriented, and high-level programming language. The language was created by Guido van Rossum and first released in 1991. Python's design emphasizes code readability, notably through the use of significant whitespace. Python supports modules and packages, which allows for code reuse. Python is a general-purpose programming language, which means it can be used for almost anything.
